Readers¶

Factory¶

cutevariant.core.readerfactory.create_reader(filepath)[source]¶

Context manager that wraps the given file and return an accurate reader

A detection of the file type is made as well as a detection of the annotations format if required.

Filetypes and annotations parsers supported:

  • vcf.gz: snpeff, vep

  • vcf: snpeff, vep

  • csv, tsv, txt: vep

cutevariant.core.readerfactory.detect_vcf_annotation(filepath)[source]¶

Return the name of the annotation parser to be used on the given file

Called: In the importer and in the project wizard to display the detected annotations.

Returns

“vep”, “snpeff”, None
