Command line usage¶

Cutevariant cli mode helps to run actions directly from command-line.

The env variable $CUTEVARIANT_DB can be used to define a database instead of the arguments.

usage: cutevariant-cli [-h] [-vv [{debug,info,critical,error,warning}]]
                       {createdb,show,remove,exec} ...

Positional Arguments¶

subparser

Possible choices: createdb, show, remove, exec

Named Arguments¶

-vv, --verbose

Possible choices: debug, info, critical, error, warning

Default: “error”

Sub-commands:¶

createdb¶

Build a SQLite database from a vcf file

cutevariant-cli createdb [-h] [--db DB] -i INPUT

Named Arguments¶

--db

SQLite database. By default, $CUTEVARIANT_DB is used.

-i, --input

VCF file path

Examples:

$ cutevariant-cli createdb -i “examples/test.snpeff.vcf”

show¶

Display table content

cutevariant-cli show [-h] [--db DB] {fields,selections,samples,wordsets}

Positional Arguments¶

table

Possible choices: fields, selections, samples, wordsets

Possible names of tables.

Named Arguments¶

--db

SQLite database. By default, $CUTEVARIANT_DB is used.

remove¶

remove selection

cutevariant-cli remove [-h] [--db DB] names [names ...]

Positional Arguments¶

names

Name(s) of selection(s).

Named Arguments¶

--db

SQLite database. By default, $CUTEVARIANT_DB is used.

exec¶

Execute a VQL statement.

cutevariant-cli exec [-h] [--db DB] [-l LIMIT] [-s TO_SELECTION] vql

Positional Arguments¶

vql

A VQL statement.

Named Arguments¶

--db

SQLite database. By default, $CUTEVARIANT_DB is used.

-l, --limit

Limit the number of lines in output.

Default: 100

-s, --to-selection

Save SELECT query into a selection name.

Examples:

$ cutevariant-cli exec “SELECT favorite,chr,pos,ref,alt FROM variants” or $ cutevariant-cli exec “SELECT chr,ref,alt FROM variants” -s myselection or $ cutevariant-cli exec “IMPORT WORDSETs ‘examples/gene.txt’ AS mygenes” or $ cutevariant-cli exec “DROP WORDSETS mygenes” or $ cutevariant-cli exec “CREATE myselection1 FROM variants WHERE gene = ‘CHID1’” $ cutevariant-cli exec “CREATE myselection2 FROM variants WHERE gene = ‘CICP23’” $ cutevariant-cli exec “CREATE myselection3 = myselection2 | myselection2” or $ cutevariant-cli exec “CREATE boby FROM variants INTERSECT ‘examples/test.bed’”

Examples:

$ cutevariant-cli show –db my_database.db samples or $ export CUTEVARIANT_DB=my_database.db $ cutevariant-cli show samples
