cutevariant.core.writer.abstractwriter.AbstractWriter¶

class cutevariant.core.writer.abstractwriter.AbstractWriter(device)[source]¶

Bases: object

Base class for all Writer required to export variants into a file or a database.

Subclass it if you want a new file writer.

device¶

a file object typically returned by open(“w”)

Example

...    writer = MyWriter(file)
...    writer.save(conn)
