parse_fields(fields)[source]¶

Generate fields description

Called by a reader when its get_fields() method is called and when an annotation parser is set.

This function parses special annotation field “ann”/”ANN”, other fields are yielded without being affected.

See also

handle_descriptions()

Input example:

({
'name': 'generic_field1',
'description': ...
'category': ...
'type': ...
},
{
'name': 'ann',
'description': '... annotation_field_name1 | annotation_field_name2 ...',
...
})

Output example:

({
'name': 'generic_field1',
'description': ...
'category': ...
'type': ...
},
{
'name': 'annotation_field_name1',
'description': ...
'category': ...
'type': ...
},
{
'name': 'annotation_field_name2',
'description': ...
'category': ...
'type': ...
})

Note

Names of fields are changed to lowercase.
